# Bacheca Annunci

Applicazione in Java per la gestione di una bacheca di annunci, sviluppata come progetto universitario (Università del Piemonte Orientale). Il sistema gestisce un singolo tipo di utente e permette l'interazione sia tramite interfaccia grafica che da riga di comando.

## 🛠️ Tecnologie Utilizzate
* **Linguaggio:** Java
* **Architettura:** Separazione logica tra Modello, Interfaccia e Main
* **Persistenza Dati:** File system locale (bacheca.txt)

## ✨ Funzionalità
* **Gestione Annunci:** Creazione e visualizzazione degli annunci in bacheca.
* **Gestione Utente:** Sistema basato su un'unica tipologia di utente.
* **Doppia Interfaccia:**
  * **GUI:** Interfaccia grafica.
  * **CLI:** Interfaccia a riga di comando per terminale.
* **Gestione Errori:** Eccezioni personalizzate per un controllo robusto.

## 🚀 Come avviare il progetto

1. Clona la repository:
   ```bash
   git clone [https://github.com/Ricky-7-7/Bacheca_UPO.git](https://github.com/Ricky-7-7/Bacheca_Annunci_UPO.git)
   ```

2. Entra nella cartella dei sorgenti:
   ```bash
   cd Bacheca_UPO/src
   ```

3. Compila e avvia il file Main:
   ```bash
   javac main/Main.java
   java main.Main
   ```
